Installation
------------

1.  Checkout the following Subversion repository:
    http://pyreplib.googlecode.com/svn/trunk/

2a. python setup.py install

2b. (if 2a fails)
    - Start Visual Studio command line
    - Follow the instructions here: https://code.google.com/p/pyreplib/issues/detail?id=2

3.  copy the directory build\...\pyreplib\ to this directiory
